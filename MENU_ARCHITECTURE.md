# SemiPLM 菜单与功能边界

本文档记录菜单分组和业务边界。开发页面前先确认功能属于哪个菜单域，再到 `FILE_INDEX.md` 查对应代码文件。

## 菜单原则

- 菜单顺序按业务使用路径排列，基础平台后置，避免系统一打开就像配置后台。
- 菜单按业务域组织，不按页面数量凑菜单。
- 独立菜单只给高频业务对象、平台配置中心和跨对象工作台。
- 版本、签核、发布、发放、回收、预览、权限等是对象内部能力，不单独做菜单。
- 分类、属性模板、生命周期、编码规则、数据字典、系统参数属于基础平台。
- ERP/MES/QMS 接口配置和同步队列分开：配置归基础/接口配置，队列归集成中心。
- 产品版本、产品资料包优先在产品库/产品详情呈现，不拆独立菜单。
- BOM 比较、反查、审核发布、导入导出、批量编辑是 BOM 页面内部能力。
- 流程节点、流程适用范围、流程日志属于流程配置或流程实例详情，不拆碎菜单。
- 当前只开放已有真实页面，不放空壳入口。

## 当前左侧菜单

### 1. 工作台

- 研发驾驶舱：产品、项目、BOM、文档、质量、变更、集成概览。
- 我的待办：流程任务、我的变更、我的项目、我的任务、我的质量问题、阶段门风险、近期动态、任务日历、消息通知。
- 闭环验证：按产品检查需求、版本、BOM、工艺、文档、变更、项目、质量、集成断点。

### 2. 产品中心

- 产品库：芯片型号、产品名称、工艺平台、封装、晶圆尺寸、生命周期、产品详情聚合。
- 需求规格：客户规格、质量体系输入、NPI 阶段门需求、验收准则、追溯链路。
- 物料库：研发物料、原材料、辅料、治具、供应商、生命周期。
- 替代料管理：替代料关系、替代策略、生效范围、风险等级。
- 供应商/制造商：供应商、制造商、认证状态、供货风险。

### 3. 文档中心

- 文档库：文档 CRUD、提交发布、版本历史、预览、发放回收、附件。

### 4. BOM 管理

- 设计 BOM：EBOM/PBOM/MBOM、明细、有效性、比较、反查、导入导出、批量编辑、版本历史、发布。
- BOM 基线：产品发布包冻结产品、BOM、文档、工艺、变更。

### 5. 工艺管理

- 工艺路线：路线、工序、工序物料绑定、提交发布、MES 队列、版本历史。
- 工艺参数库：CD、Overlay、刻蚀深度、膜厚、片阻、LIV/IV、Wafer map 等参数定义。

### 6. 工程变更

- PR 问题报告：问题来源、描述、严重度、建议处理方式。
- ECR/ECO/ECN：变更申请、变更指令、变更通知、影响分析、ECA 动作、对象升版、生效批次、下游同步。

### 7. 项目管理

- 项目库：项目、模板、任务、交付物、风险、阶段门、甘特图。

### 8. 质量闭环

- 质量问题：质量问题、CAPA、质量报告、Lot/Wafer 追溯、质量触发 ECR。

### 9. 集成中心

- 同步队列：ERP/MES/QMS 队列、状态、处理、失败、重试、对象同步记录。
- 接口配置：系统、地址、方向、认证、对象范围、负责人。

### 10. 报表与审计

- 报表中心：数据完整度、变更周期、项目进度、质量闭环、Excel 导出、报表快照。
- 操作日志：新增、编辑、删除、审批、发布、关闭等关键记录。

### 11. 基础平台

- 用户管理：账号、姓名、角色、部门、状态。
- 角色管理：角色、权限范围、状态。
- 组织管理：公司、部门基础信息。
- 流程配置：流程模板、节点、角色、动作、SLA。
- 编码规则：产品、物料、文档、BOM、ECR/ECN/ECA、项目编码。
- 数据字典：状态、类型、优先级、风险等级、文档分类、变更类型。
- 生命周期：产品、物料、文档、BOM、工艺、变更生命周期模板。
- 分类属性：产品、物料、文档、BOM、工艺、变更分类与属性模板。
- 系统参数：公司名称、默认部门、文件存储、编号前缀、定时任务参数。

## 第一批业务闭环

第一批目标不是铺满所有菜单，而是让产品数据形成可演示、可追溯、可继续深化的主链路：

产品/需求 -> 文档 -> EBOM -> 工艺/PBOM/MBOM -> 变更 -> 发布 -> 集成队列 -> 质量/项目追溯。

第一批必须优先保证以下闭环能力：

- 产品详情能聚合需求、文档、BOM、工艺、项目、质量和变更。
- 文档、BOM、工艺具备版本、发布和关联对象追溯。
- ECO 审批后能驱动文档/BOM/工艺升版或形成明确 ECA 动作。
- 项目交付物能绑定真实文档版本、BOM 版本或质量报告。
- 质量问题能触发 ECR，并能回看 CAPA、质量报告和关联批次。
- 发布类动作进入 ERP/MES/QMS 集成队列，失败可查看并重试。
- 工作台能展示待办、风险、近期动态和闭环验证结果。

## 后续菜单缺口

短期不新增主菜单，优先在现有页面内深化对象能力：

- BOM：MBOM/PBOM 转换深化、转换血缘、工序物料分配完整性、基线追溯。
- 文档：对象关联全景、ECO 发布联动发放。
- 项目：跨模块关联展示、交付物版本绑定、项目归档。
- 工艺：工装辅料、检验标准。
- 集成：真实对接后的对象映射、字段映射、回调处理、接口日志。

## 暂缓入口

以下能力不进入当前主菜单：

- CAD/EDA 深集成。
- Office/CAD 全格式在线预览。
- 移动端。
- 数据大屏。
- 多组织/多租户管理。
